# Finance-Dashboard-UI
A simple and interactive Finance Dashboard to track income, expenses, and financial insights with charts and filtering features.


💰 Finance Dashboard

This is a frontend-based Finance Dashboard project built using HTML, CSS, and JavaScript. It helps users track their income, expenses, and overall financial activity in a simple and interactive way.

🚀 Features
- View total balance, income, and expenses
- Add, edit, and delete transactions (Admin role)
- Search and filter transactions
- Interactive charts (balance trend & spending breakdown)
- Role-based UI (Viewer/Admin)
- Responsive design for different screen sizes

🛠️ Tech Stack
- HTML
- CSS (Flexbox, Grid, responsive design)
- JavaScript (Class-based structure, DOM manipulation)
- Chart.js (for data visualization)

⚠️ Limitations
- No backend integration (data resets on refresh)
- Role system is only simulated (no real authentication)

🔮 Future Improvements
- Add backend (Node.js + database)
- Implement authentication system
- Store data permanently
- Improve performance for large datasets

🎯 Purpose
This project was built to practice core frontend development concepts and understand how a real-world dashboard works.
